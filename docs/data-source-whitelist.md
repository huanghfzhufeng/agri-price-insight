# 数据源白名单

## 1. 文档目的

本文档用于明确本项目允许使用的真实数据来源，避免后续开发过程中混入不稳定、不可追溯或不适合毕业设计答辩的数据源。

本项目的原则是：

- 价格主数据优先使用官方来源
- 分析背景优先使用官方月报
- 论文口径优先使用国家统计制度说明
- 图片资源与价格数据分层处理，不混用

## 2. 价格主数据源

### 2.1 农业农村部数据查询平台

网址：

- [https://data.moa.gov.cn/](https://data.moa.gov.cn/)

用途：

- 作为农产品价格数据的官方入口
- 优先查找可直接获取的价格、指数和历史数据

计划用途：

- 构建价格主数据表
- 构建指数趋势数据
- 作为每日/历史价格查询的补充来源

计划抓取字段：

- 日期
- 指数值
- 品种名称
- 价格值
- 单位
- 来源说明

说明：

- 这是本项目的一级官方源
- 适合作为论文中“数据来源可靠性”的主要依据

### 2.2 农业农村部每日批发市场价格简报

示例网址：

- [https://www.moa.gov.cn/xw/zxfb/202502/t20250211_6469865.htm](https://www.moa.gov.cn/xw/zxfb/202502/t20250211_6469865.htm)
- [https://www.moa.gov.cn/xw/zxfb/202502/t20250206_6469707.htm](https://www.moa.gov.cn/xw/zxfb/202502/t20250206_6469707.htm)
- [https://www.moa.gov.cn/xw/zxfb/202504/t20250403_6472771.htm](https://www.moa.gov.cn/xw/zxfb/202504/t20250403_6472771.htm)

用途：

- 作为第一阶段最容易落地的真实采集源
- 构建每日价格摘要、监测指标和涨跌提示

计划抓取字段：

- 发布日期
- 农产品批发价格 200 指数
- 菜篮子产品批发价格指数
- 重点品种价格
- 同比/环比描述文本
- 来源链接
- 原始文本

说明：

- 此类页面结构相对稳定
- 适合先实现“真实采集 -> 清洗 -> 入库 -> 页面展示”闭环

### 2.3 农业农村部市场与信息化司价格监测页

示例网址：

- [https://scs.moa.gov.cn/jcyj/202503/t20250320_6472066.htm](https://scs.moa.gov.cn/jcyj/202503/t20250320_6472066.htm)

用途：

- 作为农业农村部日度价格信息的补充源或备用源
- 当主站同类内容抓取不稳定时可作为兜底来源

说明：

- 该来源与农业农村部主站存在内容关联
- 第一阶段不作为唯一主源，只作为补充来源

## 3. 分析背景数据源

### 3.1 农业农村部农产品供需形势分析月报

示例网址：

- [https://www.moa.gov.cn/ztzl/nybrl/rlxx/202503/P020250324376108713746.pdf](https://www.moa.gov.cn/ztzl/nybrl/rlxx/202503/P020250324376108713746.pdf)
- [https://www.moa.gov.cn/ztzl/nybrl/rlxx/202501/P020250121503390297799.pdf](https://www.moa.gov.cn/ztzl/nybrl/rlxx/202501/P020250121503390297799.pdf)

用途：

- 为论文提供市场背景分析
- 为预测结果解释提供依据
- 提取月度供需、价格趋势、风险因素

计划用途：

- 展示系统“分析说明”模块
- 为预测页生成辅助说明文案

说明：

- 此类 PDF 不适合作为高频主数据表
- 适合作为月度分析层和论文材料支撑

## 4. 论文口径与制度说明来源

### 4.1 国家统计局价格调查制度说明

网址：

- [https://www.stats.gov.cn/fw/bmdcxmsp/bmzd/202303/t20230330_1938225.html](https://www.stats.gov.cn/fw/bmdcxmsp/bmzd/202303/t20230330_1938225.html)

用途：

- 用于论文写作中的“数据背景、调查范围、统计口径”说明
- 强化项目数据来源的规范性和研究背景严谨性

说明：

- 该来源不作为系统主抓取源
- 主要用于论文背景和设计合理性说明

## 5. 第一阶段批准采集策略

当前批准执行的抓取路线如下：

### 主路线

- 优先抓取农业农村部每日批发市场价格简报

### 辅路线

- 通过农业农村部数据查询平台补充指数类和结构化数据

### 分析路线

- 通过农业农村部月报提取月度分析结论

### 论文支撑路线

- 通过国家统计局制度说明补充调查口径依据

## 6. 暂不纳入白名单的数据类型

以下数据在当前阶段不纳入正式白名单：

- 来源不清晰的商业站点价格数据
- 无法验证授权的图片搜索结果页面
- 社交媒体截图数据
- 未说明采集口径的自媒体行情汇总页

## 7. 开发执行约束

后续开发必须遵循：

- 采集脚本只从本白名单来源获取价格主数据
- 如果新增来源，必须先更新本文档
- 如果页面结构失效，应优先切换到白名单中的备用源

## 8. 当前结论

本项目当前已批准的真实数据来源足以支撑：

- 价格监测
- 趋势分析
- 预测解释
- 论文数据背景说明

- 原始表与标准表
