# 系统使用说明

## 1. 登录系统

默认管理员账号：

- 用户名：`admin`
- 密码：`Admin@123456`

## 2. 系统概览

可查看：

- 今日更新数量
- 监控品类数量
- 异常预警数量
- 重点品类趋势
- 涨跌排行

## 3. 数据查询

支持按以下条件筛选：

- 农产品名称
- 市场名称
- 分类
- 开始日期
- 结束日期

支持功能：

- 分页浏览
- CSV 导出
- 产品图片展示

## 4. 统计分析

可查看：

- 最新价格
- 30 日均价
- 同比与环比
- 区域对比
- 波动率排行
- IQR 异常识别

## 5. 预测预警

可查看：

- 当前预警列表
- 7 天 / 30 天 / 90 天预测
- 置信区间
- 模型评价指标
- 模型对比标签

## 6. 系统管理

可查看或操作：

- 采集任务日志
- 原始采集文章
- 数据源管理
- 阈值配置
- 月报归档

## 7. 推荐答辩演示路线

1. 登录系统
2. 展示系统概览和涨跌榜
3. 进入数据查询页筛选“大蒜”
4. 进入统计分析页展示同比环比与区域对比
5. 进入预测预警页展示模型结果
6. 进入系统管理页展示真实采集日志和月报归档
